# Configurations

This is my personal `dotfiles` repository. I use the tool
[chezmoi](https://chezmoi.io/) to manage my configurations across multiple machines.

## Setup

Install `chezmoi`; e.g. on MacOS with Homebrew:

```sh
brew install chezmoi
```

Pull the dotfiles:

```sh
chezmoi init N8WM
```
